# ✨ Neon Tic-Tac-Toe

Современная браузерная версия **Tic-Tac-Toe** с неоновым интерфейсом, AI-соперником, темами оформления и сохранением прогресса.

---

## 🚀 Возможности

- 🎮 **2 режима**: Player vs Player и Player vs AI
- 🧠 **AI на minimax** (играет за `O`)
- 💾 **Сохранение в localStorage**:
  - счёт матча
  - выбранный режим
  - выбранная тема
- 🎨 **Темы**: Neon / Sunset / Midnight
- 🔊 **Микро-звуки**: ход, победа, ничья
- ✨ **Микро-анимации**: pop-эффект на последнем ходе + подсветка победной линии
- 📱 Адаптивный UI для мобильных и десктопа

---

## 🧩 Структура проекта

```text
.
├── index.html    # UI + режимы + выбор темы
├── styles.css    # Неон/темы/анимации
├── script.js     # Логика игры, AI(minimax), localStorage, sound
└── README.md
```

---

## ⚡ Запуск

Открой `index.html` в браузере

или:

```bash
python3 -m http.server 8080
```

Перейди на `http://localhost:8080`
